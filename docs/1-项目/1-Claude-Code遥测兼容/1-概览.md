# Claude Code 遥测兼容项目概览

## 目标结果

为插件增加显式的 Claude Code 遥测兼容 Profile，使 opencode 产生的 Metrics、Logs/Events 和 Traces 能被面向 Claude Code Schema 的 SigNoz Dashboard、查询和告警直接消费。

兼容目标是遥测 Schema 和查询语义，不是伪造 Claude Code 客户端、Anthropic 服务或用户身份。插件保留真实的 `service.name`、版本、instrumentation scope 和可验证的数据来源；无法从 opencode 获得的字段不生成。

## 范围

- 通过显式配置启用 Claude Code 兼容 Profile，默认 Profile 保持现有行为。
- 对齐 Claude Code 的核心 Metric 名称、单位、属性和枚举。
- 对齐核心 Log/Event 的 body、`event.name`、关联字段和默认脱敏行为。
- 对齐 `interaction`、`llm_request` 和 `tool` Trace 层级与主要属性。
- 提供兼容矩阵、契约测试和 SigNoz 端到端验证方法。
- 不伪造 Anthropic 账号、组织、邮箱、Claude Code 版本或官方 instrumentation scope。
